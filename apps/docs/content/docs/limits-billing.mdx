## Pricing

| Tier | Price | Includes |
|------|-------|----------|
| `bsk_test_` | Free | Full API, integration testing |
| `bsk_live_` | **$0.10 / successful parse** | Production, metered monthly via Stripe |
| Self-host | Free | Run the AGPL-3.0 stack yourself |

## What's billed

Only **successful** parses from a `live` key. Parse failures (any 4xx/5xx) are never billed — if the engine can't read it, you don't pay. CSV and redact requests bill the same as a JSON parse (one operation = one charge).

## Rate limits

The anonymous demo surface is capped at **10 parses/hour per IP**. Authenticated keys are not hard-rate-limited during the alpha; abuse protection scales with load. A `429` returns the standard envelope with `error_class: "RateLimitError"`.

## Usage

`GET /v1/usage` returns the current calendar-month parse count and a projected invoice for the calling key.
